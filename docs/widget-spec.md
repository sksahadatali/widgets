# Widget Specification

Every widget follows this structure:

``` text
widget-name/
│
├── index.html
├── config.js
├── widget.js
└── README.md
```

## Required Properties

-   Title
-   Icon
-   Primary Value
-   Secondary Value
-   Refresh Interval

## Behaviour

-   Responsive
-   Mobile friendly
-   Dark mode
-   Lightweight
-   Configurable

## Design

-   Shared typography
-   Shared spacing
-   Shared colours
-   Shared icons

Business logic belongs inside Services, not inside widgets.
