# eY Widgets Vision

## Mission

Empower people to build beautiful, lightweight, configurable dashboards
for everyday life.

The initial focus is creating reusable widgets for Notion dashboards.
The long-term vision is to evolve into a reusable dashboard framework
for Notion, Home Assistant and web applications.

## Principles

-   Simple
-   Fast
-   Beautiful
-   Configurable
-   Reusable
-   Open architecture

## Primary Target

A personal dashboard combining Family, Property, Business, Finance and
Productivity.

## Future Vision

-   Public widget library
-   Dashboard builder
-   Mobile application
-   SaaS platform
