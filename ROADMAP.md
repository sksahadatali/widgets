# eY Widgets Roadmap

> **eY Widgets** is a lightweight JavaScript framework for building reusable, embeddable dashboard widgets and applications.

---

# Vision

Build the best modular widget framework for creating personal, business and smart-display dashboards.

The framework should be:

- Modular
- Reusable
- Configuration-driven
- API-first
- Themeable
- Extensible
- Easy to embed (Notion, websites, kiosks, touch displays)

---
