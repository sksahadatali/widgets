# User Journey

Most users will follow this daily sequence:

Morning

Home
→ AI Brief
→ Today's Focus
→ Daily
→ Personal / RAEN / AYANOH
→ End of Day Review

---

# Design Principle

Navigation should require no more than three clicks to reach any destination.
