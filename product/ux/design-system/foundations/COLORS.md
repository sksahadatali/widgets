# eY OS Design System
## Colors v1.0

Status: Frozen

Purpose:
Defines the colour palette used across eY OS to ensure a consistent, premium and accessible user experience.

---

# Primary Palette

| Token | Hex | Usage |
|-------|------|------|
| bg-page | #131C31 | Main application background |
| bg-sidebar | #0F172A | Sidebar background |
| bg-card | #1A2438 | Cards and panels |
| bg-hover | #24324A | Hover state |
| bg-active | #2C3B57 | Active navigation |

# Rules

• Use bg-page for the application background.
• Sidebar always uses bg-sidebar.
• Cards always use bg-card.
• Gold is reserved for emphasis and active states.
• Never use more than one accent colour inside the same component.
