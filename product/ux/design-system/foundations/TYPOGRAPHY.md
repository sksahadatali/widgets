# Rules

• Never use more than three font sizes within one card.
• Every card must have a clear visual hierarchy.
• Use white for primary information.
• Use muted grey for supporting information.
• Gold is reserved for emphasis, active states and headings.
• Avoid bold text unless highlighting something important.
