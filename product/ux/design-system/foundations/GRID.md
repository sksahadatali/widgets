# Alignment

• All modules align to the same left edge.

• All modules align to the same right edge.

• Section headings align with the left edge of modules.

• Cards maintain equal spacing horizontally and vertically.

• Follow the 8px spacing system.

---

# Layout Principles

1. Sidebar remains fixed.

2. Header remains fixed.

3. Main content scrolls independently.

4. AI Morning Brief is always the first module on the Home page.

5. Every page follows the same layout framework.

6. New modules must follow the spacing and alignment rules defined in this document.
